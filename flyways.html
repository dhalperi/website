SIGCOMM 2011 release of Flyways Code

This code has not changed since we wrote the paper, and is provided as-is to any interested party.

Here is a tgz of the flyways ns-3 directory. You can build the code with ./build.py, and our executables are mostly in the ns3/scratch directory.

Some uers have had issues building the code because they use a newer version of GCC than at the time of development. I recommend the ns-3 documentation: https://www.nsnam.org/wiki/index.php/HOWTO_build_old_versions_of_ns-3_on_newer_compilers.
